## 开服第二周付费
```
SELECT
    o.server_id,
    o.uid,
    /* 金额换算并保留2位小数 */
    ROUND(SUM(o.amount / 100), 2) AS total_recharge
FROM
    db_ro3_sdk2.T_ORDER o
INNER JOIN
    db_ro3_server.T_SERVER s ON o.server_id = s.id
WHERE
    /* 1. 筛选充值成功 (根据你的描述 status=2) */
    o.status = 2
    
    /* 2. 筛选服务器范围 (如果你只要特定服务器，请保留此行，否则可删除) */
    AND o.server_id BETWEEN 40180 AND 40196
    
    /* 3. 动态时间筛选：开服第2周 (自然天) */
    /* 
       逻辑：
       比如 12月1日开服，DATE(open_time) = 12-01
       起始 >= 12-01 + 7天 = 12-08 00:00:00
       结束 <  12-01 + 14天 = 12-15 00:00:00 (这样就刚好包含了12-14这一整天)
    */
    AND o.create_time >= DATE_ADD(DATE(s.open_time), INTERVAL 7 DAY)
    AND o.create_time <  DATE_ADD(DATE(s.open_time), INTERVAL 14 DAY)
GROUP BY
    o.server_id,
    o.uid
HAVING 
    total_recharge > 0
ORDER BY
    o.server_id,
    total_recharge DESC;
```
## 历史充值数据
```
SELECT
    server_id,
    uid,
    ROUND(SUM(amount / 100), 2) AS total_recharges
FROM
    T_ORDER
WHERE
    server_id BETWEEN 40199 AND 40205
    AND status = 2
GROUP BY
    server_id,
    uid
ORDER BY
    server_id,
    total_recharges DESC;
```
